## 🚀 **Quick Start Guide**

### 📋 **Prerequisites**
```bash
# Check if you have the required versions
node --version  # >= 18.0.0
pnpm --version  # >= 8.0.0
```

### ⚡ **Installation**
```bash
# Clone the repository
git clone https://github.com/yourusername/formcraft.git
cd formcraft

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your configuration

# Start development server
pnpm dev
```

### 🏗️ **Build & Deploy**
```bash
# Build for production
pnpm build

# Start production server
pnpm start

# Run tests
pnpm test

# Type checking
pnpm type-check

# Linting
pnpm lint
```

## 🎯 **Features Deep Dive**

<details>
<summary><b>🖱️ Drag & Drop Form Builder</b></summary>

- **Visual Interface**: Intuitive drag-and-drop interface for form creation
- **Component Library**: Rich set of form components (inputs, dropdowns, checkboxes, etc.)
- **Real-time Preview**: See your form update in real-time as you build
- **Responsive Design**: Forms automatically adapt to different screen sizes
- **Custom Styling**: Apply custom themes and styles to match your brand

</details>

<details>
<summary><b>🧩 Multi-Step Forms</b></summary>

- **Step Management**: Create complex multi-step forms with ease
- **Progress Tracking**: Visual progress indicators for users
- **Conditional Logic**: Show/hide steps based on user responses
- **Navigation Controls**: Flexible navigation between steps
- **Data Persistence**: Save progress across sessions

</details>

<details>
<summary><b>⚡ Advanced Validation</b></summary>

- **Real-time Validation**: Validate inputs as users type
- **Custom Rules**: Create complex validation rules
- **Error Handling**: Beautiful error messages and handling
- **Schema Validation**: Powered by Zod for type-safe validation
- **Async Validation**: Support for server-side validation

</details>

<details>
<summary><b>📊 Analytics & Insights</b></summary>

- **Submission Tracking**: Monitor form submissions and completion rates
- **User Behavior**: Analyze how users interact with your forms
- **Performance Metrics**: Track form loading times and user experience
- **Export Data**: Export form data in various formats
- **Real-time Dashboard**: Live updates on form performance

</details>
